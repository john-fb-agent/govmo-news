## 🔧 技術選型（待決定）

### 選項 1：靜態網站（推薦用於 MVP）

- **框架：** Vue.js / React / 純 HTML+JS
- **部署：** GitHub Pages / Netlify
- **優點：** 簡單，快速，低成本
- **缺點：** 功能有限

### 選項 2：動態網站

- **框架：** Flask / FastAPI + Vue/React
- **部署：** Docker / VPS
- **優點：** 功能完整，可擴展
- **缺點：** 複雜度高

### 選項 3：Jupyter Notebook（快速原型）

- **用途：** 數據探索和原型展示
- **優點：** 快速，適合數據分析
- **缺點：** 不適合生產環境
