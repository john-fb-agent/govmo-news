# 任務
分析澳門政府新聞 JSON 數據，生成 HTML 格式的每日新聞總結。

# 輸入數據
{news_json}

# 處理規則

## 1. AI 自動分類
根據新聞標題和摘要，自動分類到：政策、民生、經濟、旅遊、教育、文化、科技、其他

## 2. 重要性評分
**⭐⭐⭐ 高重要性**（需要摘要）：新政策、重大建設、影響廣泛民生、突發重要事件
**⭐⭐ 中重要性**（只需標題）：常規政策、部門活動、數據發布
**⭐ 低重要性**（只需標題）：常規會議、小型活動、人事任命

## 3. 輸出要求
- 高重要性新聞不超過 5 則
- 只輸出 JSON 格式，不要其他文字

# 輸出格式
```json
{
  "date": "2026 年 4 月 14 日",
  "total_count": 15,
  "highlights": [
    {
      "title": "新聞標題",
      "category": "政策",
      "importance": 3,
      "summary": "1-2 句話摘要",
      "link": "https://..."
    }
  ],
  "category_stats": {"政策": 3, "民生": 5, "經濟": 2, "其他": 5},
  "all_news": [
    {
      "title": "新聞標題",
      "category": "民生",
      "importance": 2,
      "link": "https://..."
    }
  ]
}
```

# 注意事項
- 使用繁體中文
- 只輸出 JSON，不要 Markdown 或其他文字
- 高重要性新聞（importance=3）才需要 summary 字段
