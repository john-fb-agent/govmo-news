## 🟢 輕微問題

### 1. 日誌文件（已解決）

**現狀：** 日誌文件已加入 `.gitignore`，不提交到 Git

**影響：**
- ✅ 不影響 Git 倉庫大小
- ⚠️ 本地運行時仍會產生日誌

**緩解措施：**
- 手動清理：`rm data/*.log`

### 2. 無監控告警

**問題：** 抓取失敗時無自動通知

**影響：**
- 可能長時間未發現抓取失敗

**緩解措施：**
- 定期檢查日誌
- 檢查 `data/processed/` 是否有新數據

**計劃：** 添加監控和告警機制

**追蹤 Issue：** 待建立

## 📝 報告新問題

如發現新問題，請：

1. **檢查是否為已知問題** - 查看本文檔
2. **收集信息**：
   - 錯誤訊息
   - 重現步驟
   - 環境信息（OS、Python 版本等）
   - 日誌片段
3. **提交 Issue**：https://github.com/john-fb-agent/govmo-news/issues
