## [1.3.0] - 2026-04-15

### Added
- ➕ **GitHub Pages 新聞總結功能**
  - 每天早上 8 點自動生成前日新聞總結
  - AI 自動分類（政策、民生、經濟、教育、文化、科技、旅遊、其他）
  - AI 自動評分重要性（高/中/低）
  - 生成 500 字綜合摘要
  - 選出最多 5 則重點新聞
  - 自動提交並推送到 GitHub
  - Telegram 通知執行結果
- 🌐 **GitHub Pages 部署**
  - 索引頁面：https://john-fb-agent.github.io/govmo-news/summaries/
  - 每日總結：https://john-fb-agent.github.io/govmo-news/summaries/YYYY-MM-DD.html
  - 自動維護新聞列表
- 🤖 **OpenClaw Cron 自動化**
  - 使用 isolated session 執行長時間任務
  - 自動錯誤處理和通知
  - 支援超時重試機制

### Changed
- 📁 新聞總結存儲位置：`docs/summaries/`（GitHub Pages 兼容）
- 📝 Footer 添加 AI Provider 和 Model 資訊（Qwen / qwen3.5-plus）
- 🔢 版本號更新為 1.3.0

## [1.2.0] - 2026-04-15

### Fixed
- 🐛 **修復 JSON 文件覆蓋 bug**
  - 問題：每次抓取都會覆蓋當天的 JSON 文件，導致一天內多次抓取時只有最後一次的新聞被保存
  - 解決：改為合併模式（載入現有 → 合併新新聞 → 保存）
  - 影響：現在每天只有一個 JSON 文件，會累積該日所有新聞（不會覆蓋）

### Changed
- 📝 更新所有文檔以反映修復
- 🔢 版本號更新為 1.2.0

## [1.1.0] - 2026-04-15

### Added
- ➕ **添加自動推送功能**
  - 新增 `auto-push.sh` 腳本
  - 每晚 10 點（22:00）自動推送新聞數據到 GitHub
  - 自動推送只執行 commit + push，不抓取新聞（data 已在 17:00 抓取）
  - 新增 `setup-auto-push.sh` 和 `remove-auto-push.sh` 腳本

### Changed
- 📝 更新部署文檔
- 🔢 版本號更新為 1.1.0
