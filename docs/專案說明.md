# 專案說明

## 📋 專案概述

**專案名稱：** govmo-news（澳門政府新聞抓取系統）

**專案目標：** 自動抓取澳門政府新聞局（GCS）RSS 新聞，轉換為 JSON 格式，按日期組織存儲，並提供去重功能。

## 🎯 核心功能

### 1. RSS 新聞抓取
- 從 `https://govinfohub.gcs.gov.mo/api/rss/n/zh-hant` 抓取新聞
- 支援繁體中文新聞
- 自動解析 RSS XML 格式

### 2. XML 轉 JSON
- 將 RSS XML 轉換為結構化 JSON
- 保留新聞標題、連結、發布時間、摘要、guid 等資訊
- 使用 RSS guid 作為唯一 ID 進行去重

### 3. 按日期組織
- JSON 檔案按 `年/月` 目錄結構存儲
- 格式：`data/processed/YYYY/MM/DD.json`（每天一個 JSON 檔）
- 每個 JSON 檔包含該日所有新聞（array 格式）

### 4. 自動去重
- 使用 RSS guid 進行去重
- 避免重複保存相同新聞
- 永久保存所有新聞數據

### 5. 定時任務
- **新聞抓取：** 每日執行 5 次：09:00, 11:00, 13:00, 15:00, 18:00（澳門時間 GMT+8）
- **自動抓取 + 推送：** 每日執行 1 次：01:00（凌晨 1 點）
- 推送前先執行抓取，確保數據是最新的
- 使用 cron 調度
- 自動記錄執行日誌

```

    ↓
```

## 🔧 技術選型

### 語言與框架
- **Python 3.8+** - 主要開發語言
- **feedparser** - RSS 解析庫
- **requests** - HTTP 請求庫

### 數據存儲
- **JSON 檔案** - 新聞數據存儲
- **檔案系統** - 按日期組織目錄結構

### 調度
- **cron** - Linux 定時任務

### 為什麼選擇這些技術？

| 技術 | 選擇理由 | 替代方案 |
|------|---------|---------|
| Python | 簡單易用，RSS 解析庫成熟 | Node.js, Go |
| JSON | 通用格式，易於處理 | XML, CSV |
| cron | Linux 標準，簡單可靠 | systemd timers, Airflow |
| 檔案系統 | 簡單，無需數據庫 | MongoDB, PostgreSQL |
